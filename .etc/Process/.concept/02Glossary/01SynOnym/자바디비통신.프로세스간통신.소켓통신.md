- 자바디비통신👉통신의 핵심: TCP/IP 소켓
- 설령 같은 PC에 떠 있더라도, 자바는 DB가 열어놓은 특정 포트(예: MySQL은 3306, PostgreSQL은 5432)를 통해 **TCP/IP 소켓 연결**을 맺고 데이터를 주고받습니다.
- Java 애플리케이션과 데이터베이스(DB) 사이의 통신은 넓은 의미에서 IPC에 해당하지만, 실제로는 **소켓(Socket) 통신**을 기반으로 한 **클라이언트-서버 모델**로 동작합니다.
