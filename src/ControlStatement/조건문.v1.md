### **📌 조건문을 표현하는 개발 용어**  

1. **Control Flow (제어 흐름)**  
   - 코드 실행 흐름을 제어하는 방식. 조건문, 반복문, 예외 처리 등이 포함됨.  
   - 예: `if-else`, `switch-case`, `ternary operator (삼항 연산자)` 등  

2. **Conditional Statement (조건문)**  
   - `if`, `else if`, `else`, `switch` 같은 직접적인 조건문을 의미  
   - 예:  
     ```js
     if (x > 10) { 
         console.log("10보다 큼");
     } else {
         console.log("10 이하");
     }
     ```

3. **Branching (분기문)**  
   - 코드가 여러 갈래로 나뉘어 실행되는 것을 의미  
   - 예: `if-else`, `switch-case`  

4. **Guard Clause (가드 절, 조기 반환)**  
   - 조건이 맞지 않으면 빠르게 리턴(return)해서 함수의 깊이를 줄이는 패턴  
   - 예:  
     ```js
     function processUser(user) {
         if (!user) return;  // 사용자가 없으면 즉시 종료
         console.log(user.name);
     }
     ```

5. **Ternary Operator (삼항 연산자)**  
   - 짧은 조건문을 한 줄로 작성할 때 사용  
   - 예:  
     ```js
     let result = age >= 18 ? "성인" : "미성년자";
     ```

6. **Short-Circuit Evaluation (단축 평가)**  
   - 논리 연산자 (`&&`, `||`)를 이용하여 조건문을 줄이는 방법  
   - 예:  
     ```js
     user && console.log(user.name);  // user가 있으면 실행
     ```

7. **Switch-Case Statement (스위치 문)**  
   - 여러 개의 경우를 처리할 때 사용  
   - 예:  
     ```js
     switch (value) {
         case 1:
             console.log("One");
             break;
         case 2:
             console.log("Two");
             break;
         default:
             console.log("Other");
     }
     ```

8. **Predicate Function (판별 함수)**  
   - `true` 또는 `false`를 반환하는 함수 (조건문과 유사한 역할)  
   - 예:  
     ```js
     function isAdult(age) {
         return age >= 18;
     }
     ```

9. **If Expression (조건식을 반환값으로 사용)**  
   - `if`문을 값으로 처리하는 패턴 (언어에 따라 지원 여부 다름)  
   - 예 (Kotlin):  
     ```kotlin
     val result = if (x > 10) "큰 값" else "작은 값"
     ```
