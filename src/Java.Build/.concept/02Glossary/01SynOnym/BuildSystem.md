- 빌드시스템
- Build System
- BuildSystem
- "프로젝트를 조립하는 기계(엔진)"라고 생각하면 쉽습니다. 코드를 컴파일하고, 라이브러리를 가져오고, 실행 파일(JAR)을 만드는 전체 공정을 담당합니다.
- 종류: IntelliJ, Maven, Gradle
- IntelliJ: 인텔리제이 자체 엔진을 사용합니다. 설정이 매우 빠르지만, 팀원들과 환경을 공유하기 어렵고 복잡한 빌드 설정을 하기엔 한계가 있습니다.
- Gradle: 현재 안드로이드 및 자바 백엔드에서 가장 많이 쓰이는 표준 기계입니다. 유연하고 강력하며, 어떤 컴퓨터에서든 동일한 빌드 결과를 보장합니다.
