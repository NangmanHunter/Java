- 일반적으로 우리가 그냥 '프록시'라고 부를 때의 의미입니다.
- 우리가 인터넷을 사용할 때의 기본 흐름은 [클라이언트 -> 서버] 방향입니다.
- 누구를 대신하나: 클라이언트(사용자)를 대신합니다.
- 왜 쓰나: 클라이언트가 누군지 서버가 모르게 하거나(IP 세탁), 특정 사이트 접근을 막거나, 자주 가는 사이트 데이터를 캐싱해서 빨리 보여주려고 씁니다.
- 비유: 학교나 회사에서 학생들이 딴짓 못 하게 막는 '감시 필터' 역할을 하는 대리인과 같습니다.
- 포워드 프록시는 클라이언트와 같은 방향(앞쪽)에 서서 서버를 바라봅니다.
- 프록시(Forward): "내가 누군지 서버한테 비밀로 해줘" (클라이언트 보호)
