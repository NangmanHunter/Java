



### 주요 차이점 요약

| 구분 | Prototype (프로토타입) | MVP (최소 기능 제품) |
| :--- | :--- | :--- |
| **핵심 목적** | 아이디어 시각화 및 기술적 검증 | 시장성 검증 및 실제 사용자 가치 전달 |
| **주요 대상** | 내부 팀, 투자자, 소수 테스트 그룹 | **실제 시장의 초기 사용자(Early Adopters)** |
| **완성도** | 기능이 동작하지 않을 수 있음 (껍데기) | **실제로 동작하며 결제/사용이 가능함** |
| **결과물** | 스케치, 와이어프레임, 모형 | 실제 배포 가능한 최소한의 제품 |
