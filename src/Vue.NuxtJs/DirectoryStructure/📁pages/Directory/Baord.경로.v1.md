##
- `/board`
  - 절대경로
- `./`
  - `./`
  - `../board`
  - 상대경로


##
현재 위치가 `/board/write`이고, 한 단계 상위인 `/board` 목록 페이지로 보내고 싶다면 **상대 경로**나 **절대 경로** 둘 다 사용할 수 있습니다.

상황에 맞는 가장 깔끔한 방법들을 정리해 드릴게요.

---

### 1. 가장 확실한 방법 (절대 경로)
경로를 직접 명시하는 방법입니다. 프로젝트 구조가 바뀌어도 의도가 명확해서 가장 추천합니다.

```javascript
navigateTo('/board')
```

### 2. 현재 위치 기준 (상대 경로)
현재 `/board/write`에 있으므로, 한 단계 위로 올라가는 방식입니다.

```javascript
navigateTo('../board')
// 또는 현재가 /board/ 하위임이 확실하다면
navigateTo('./') 
```

### 3. 이전 페이지로 돌아가기 (브라우저 히스토리)
만약 사용자가 목록에서 '글쓰기'를 눌러 들어온 것이 확실하고, 단순히 "뒤로 가기" 기능을 원한다면 Nuxt 내장 함수 대신 브라우저 기능을 쓸 수도 있습니다.

```javascript
// 이건 Nuxt 함수는 아니지만 브라우저 히스토리를 이용합니다.
useRouter().back()
```

---

### 💡 팁: NuxtUI의 `UButton`과 함께 쓸 때
만약 취소 버튼 같은 곳에 넣으신다면 함수를 따로 만들지 않고 템플릿에서 바로 처리할 수도 있습니다.

```vue
<UButton 
  label="취소" 
  variant="ghost" 
  @click="navigateTo('/board')" 
/>
```

**결론:** 가장 실수 없는 방법은 **`MapsTo('/board')`**를 사용하는 것입니다!
