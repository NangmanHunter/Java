- 웹 접근성(Accessibility, A11y)을 위해 존재하는 아주 기특한 태그입니다.
- 역할: 페이지가 바뀔 때(Route 전환), 시각 장애인이 사용하는 스크린 리더(Screen Reader)에게 "지금 페이지가 바뀌었습니다"라고 말해주는 역할을 합니다.
- 왜 필요한가: 일반적인 웹사이트는 페이지 이동 시 화면이 새로고침되지만, Nuxt 같은 SPA(Single Page Application) 기술은 화면 일부만 바뀌기 때문에 스크린 리더가 페이지 이동을 인지하지 못할 때가 많습니다. 이를 해결해 줍니다.
- 특징: 화면에는 아무것도 나타나지 않습니다. (Invisible)
- ` <NuxtRouteAnnouncer />`👉웹 표준과 접근성 (Accessibility)👉현대 웹 개발에서 **웹 접근성(A11y)**은 단순히 '친절함'을 넘어 일종의 표준이자 법적 준수 사항이 되어가고 있습니다.
- ` <NuxtRouteAnnouncer />`👉검색 엔진 최적화 (SEO)와의 연관성👉직접적으로 SEO 점수를 높여주는 것은 아니지만, 구글 같은 검색 엔진은 **"모든 사용자가 이용하기 편리한 구조(접근성)를 갖춘 사이트"**에 더 높은 점수를 주는 경향이 있습니다. 웹 접근성 점수가 높을수록 '건강한 사이트'로 평가받기 때문입니다.
