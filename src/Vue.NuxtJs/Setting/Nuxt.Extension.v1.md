Nuxt와 Vue 개발 효율을 200% 올려주는 **VS Code 필수 확장 프로그램(Extensions)**들을 정리해 드릴게요. 이것들만 깔아도 백엔드 개발할 때만큼 편해집니다.

---

### 1. 필수 중의 필수 (이건 무조건!)

* **Vue - Official (구 Volar)**: 
    * **역할**: Vue 3와 Nuxt 개발의 핵심 도구입니다. 코드 하이라이팅, 에러 체크, 자동 완성을 담당합니다.
    * **주의**: 기존에 사용하던 'Vetur'라는 확장이 있다면 반드시 **비활성화(Disable)** 해야 충돌이 안 납니다.
* **TypeScript Vue Plugin (Volar)**:
    * **역할**: Nuxt는 기본적으로 타입을 아주 잘 잡아줍니다. Vue 파일 안에서 타입 체크를 더 정확하게 해줍니다.

---

### 2. 생산성 향상 (코딩이 빨라짐)

* **Prettier - Code formatter**:
    * **역할**: 코드를 저장할 때마다 줄 바꿈, 세미콜론, 따옴표 등을 자동으로 정렬해줍니다. (백엔드의 'Reformat Code' 기능)
* **ESLint**:
    * **역할**: 문법적인 오류나 안 쓰는 변수 등을 실시간으로 잡아내서 빨간 줄을 그어줍니다. 실수를 방지하는 일등 공신입니다.
* **Nuxtr**:
    * **역할**: Nuxt 전용 확장 도구입니다. 클릭 몇 번으로 페이지(`pages`), 컴포넌트(`components`), 미들웨어 등을 뚝딱 만들어줍니다.

---

### 3. 있으면 삶의 질이 올라가는 것들

* **Auto Close Tag / Auto Rename Tag**:
    * **역할**: HTML 태그를 열면 자동으로 닫아주고, 앞쪽 태그 이름을 바꾸면 뒤쪽 닫는 태그 이름도 자동으로 바꿔줍니다.
* **Tailwind CSS IntelliSense** (나중에 CSS 라이브러리 쓰실 거라면):
    * **역할**: 스타일링할 때 자동 완성을 기가 막히게 해줍니다.
* **Error Lens**:
    * **역할**: 에러 메시지를 코드 줄 바로 옆에 텍스트로 띄워줍니다. 마우스를 갖다 대지 않아도 뭐가 틀렸는지 바로 알 수 있어 정말 편합니다.

---

### 💡 세팅 팁 (저장 시 자동 정렬)

설치를 다 하셨다면 VS Code의 **설정(Settings, `Ctrl + ,`)**에서 아래 두 가지만 확인해 보세요.

1.  **Editor: Format On Save**를 체크하세요. (저장할 때마다 예쁘게 정렬됩니다.)
2.  **Default Formatter**를 `Prettier - Code formatter`로 지정하세요.
