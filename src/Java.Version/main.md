## LTS
- ❌Java 8 (구형의 전설): 아직도 많은 기업에서 쓰지만, 이제는 너무 오래되었습니다. 새로 시작하는 프로젝트에는 추천하지 않습니다.
- ✅Java 11: 한동안 표준이었으나, 지금은 점점 17로 넘어가는 추세입니다.
- ✅Java 17 (현재의 대세): 가장 강력히 추천합니다. 현재 Spring Boot 3.x 버전부터는 최소 사양이 Java 17입니다. 보안과 성능, 최신 문법(Record 등)을 모두 잡은 가장 안정적인 버전입니다.
- ✅Java 21 (최신 LTS): 2023년 말에 나온 최신 장기 지원 버전입니다. 가상 스레드(Virtual Threads) 같은 혁신적인 기능이 들어있어, 성능에 민감한 개발자들이 아주 좋아합니다.
