##
인텔리제이(IntelliJ)는 Git 기능을 아주 강력하고 직관적으로 제공하기 때문에, 터미널 명령어를 입력할 필요 없이 클릭 몇 번으로 해결할 수 있습니다.

가장 대표적인 **두 가지 방법**을 알려드릴게요.

---

### 방법 1: 단축키 활용 (가장 빠름)

숙련된 개발자들이 가장 많이 쓰는 방식입니다.

1.  **커밋(Commit) 창 열기:** `Ctrl + K` (macOS: `Cmd + K`)를 누릅니다.
    * 왼쪽 사이드바에 **Commit** 창이 뜹니다.
    * 올릴 파일들을 체크(Check)하고, 하단에 **커밋 메시지**를 적습니다.
2.  **푸시(Push)까지 한 번에 하기:**
    * 하단의 `Commit` 버튼 옆의 화살표를 눌러 **`Commit and Push...`**를 선택합니다. (이게 제일 편합니다!)
3.  **최종 확인:** 팝업창이 뜨면 내용을 확인하고 오른쪽 하단 **`Push`** 버튼을 누릅니다.

---

### 방법 2: 상단 메뉴 및 아이콘 활용

마우스를 쓰는 게 편하시다면 이 방법을 사용하세요.

1.  **아이콘 클릭:** 오른쪽 상단에 있는 **체크 표시(✔️)** 아이콘이 커밋 버튼입니다.
2.  **메시지 입력:** 왼쪽 창에 변경된 파일 목록이 보이면, 내용을 확인하고 메시지를 작성합니다.
3.  **푸시 아이콘:** 커밋이 완료된 후, 체크 표시 옆에 있는 **우상향 화살표(↗️)** 아이콘을 누르면 푸시 창이 뜹니다.



---

### 💡 인텔리제이만의 꿀팁 (체크리스트)

커밋 창을 보시면 파일 목록 위에 설정(톱니바퀴 아이콘)이나 옵션들이 있는데, 인텔리제이는 커밋 직전에 다음과 같은 일을 대신 해줄 수 있습니다.

* **Reformat code:** 코드를 깔끔하게 정렬해서 올려줍니다.
* **Optimize imports:** 안 쓰는 `import` 문을 자동으로 지워줍니다.
* **Analyze code:** 코드에 오타나 잠재적인 버그가 없는지 검사해줍니다. (노란색 경고가 뜨면 확인해 보세요!)

---

### ⚠️ 만약 GitHub 로그인을 하라고 나온다면?

처음 푸시할 때 **"Log in via GitHub..."** 버튼이 포함된 창이 뜰 수 있습니다. 
1.  그 버튼을 누르면 웹 브라우저가 열립니다.
2.  GitHub 로그인을 하고 **Authorize**를 눌러 승인해 줍니다.
3.  인텔리제이로 돌아오면 자동으로 연결되어 푸시가 진행됩니다.

---

**요약하자면:** `Ctrl + K`로 메시지 쓰고, `Commit and Push` 누르면 끝입니다! 

지금 한번 테스트로 `README.md` 파일이나 간단한 주석 하나를 수정해서 올려보시겠어요? 성공하면 하단에 **"Pushed 1 commit to origin/main"**이라는 메시지가 뜰 거예요.
